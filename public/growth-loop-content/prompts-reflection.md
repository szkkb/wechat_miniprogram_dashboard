# Reflection Prompts | 反思复盘提示词

> 使用说明：这些提示词用于定期反思和认知升级。建议每周或每两周使用一次。

---

## 周度反思

### RP-001 | 周度增长反思

```markdown
请作为我的增长教练，帮我进行本周的反思。

**本周概况**：
- 学习了什么内容：
- 执行了什么增长动作：
- 遇到了什么挑战：
- 有什么意外收获：

**请引导我思考**：

1. **认知更新**
   - 本周有什么新的认知或理解？
   - 之前的哪些认知被修正了？
   - 还有什么困惑没有解决？

2. **行动评估**
   - 本周的行动与增长理论的关联是什么？
   - 哪些行动符合 Loop 思维？哪些不符合？
   - 如果重来一次，会怎么做？

3. **模式识别**
   - 看到了什么重复出现的模式？
   - 有没有发现什么"杠杆点"？
   - 什么是消耗精力但效果不大的事？

4. **下周优先级**
   - 最重要的 1-2 件事是什么？
   - 什么可以停止做？
   - 什么实验值得尝试？

请用启发式的问题引导我，而不是直接给出答案。
```

---

### RP-002 | 学习进度检查

```markdown
请帮我检查 Growth Loop 学习进度。

**我目前学到了**：
（简述已完成的学习内容）

**请帮我评估**：

1. **理解深度检查**
   请问我几个问题，测试我对已学内容的理解：
   - 关于概念理解的问题
   - 关于应用能力的问题
   - 关于关联思考的问题

2. **知识盲点识别**
   - 基于我的回答，哪些地方可能理解不够深？
   - 哪些概念之间的关联我可能没有建立？
   - 建议我复习或深入哪些部分？

3. **下一步学习建议**
   - 接下来应该学什么？
   - 学习节奏是否需要调整？
   - 有什么额外资源可以推荐？
```

## 月度反思

### RP-101 | 月度增长复盘

```markdown
请作为增长顾问，帮我进行本月的深度复盘。

**本月数据概览**：
- 新增客户数：
- 客户来源分布：
- 收入/货量：
- 关键指标变化：

**本月主要动作**：
（列出本月执行的主要增长动作）

**请帮我深度分析**：

## 1. 结果分析
- 与上月相比，哪些指标改善了？哪些退步了？
- 增长的主要驱动因素是什么？
- 有哪些结果出乎意料？

## 2. Loop 运转评估
- 我们的核心 Loop 这个月运转得如何？
- Loop 的各个环节转化率变化如何？
- Loop 在加速还是减速？为什么？

## 3. 实验与学习总结
- 本月做了哪些实验？结果如何？
- 最大的学习收获是什么？
- 哪些假设被验证或推翻？

## 4. 资源投入 ROI
- 时间和精力主要投在哪里？
- 投入产出比最高的是什么？
- 什么是低效的，应该停止？

## 5. 下月计划
- 基于本月学习，下月的重点是什么？
- 需要做什么调整？
- 设定什么目标？

请在分析中帮我建立数据和行动之间的因果联系。
```

### RP-102 | 心智模型更新

```markdown
请帮我检查和更新关于增长的心智模型。

**我目前对增长的理解**：
（用自己的话描述你对 Growth Loop 的理解）

**过去一个月的实践经验**：
（简述实践中的关键经历和观察）

**请帮我思考**：

1. **模型检验**
   - 我描述的心智模型是否准确？
   - 有没有什么误解或偏差？
   - 实践经验是否支持这个模型？

2. **模型完善**
   - 基于实践，模型需要怎样修正？
   - 有什么新的元素应该加入？
   - 哪些假设需要修改？

3. **特殊情况**
   - [你的行业]有什么特殊性，我的模型没有考虑到？
   - 有没有遇到理论与实践不符的情况？
   - 这些特殊情况如何整合到我的理解中？

4. **下一阶段关注点**
   - 我的理解中最薄弱的环节是什么？
   - 应该通过什么方式加强？
   - 需要什么实践来验证？
```

## 季度反思

### RP-201 | 季度战略回顾

```markdown
请作为战略顾问，帮我进行季度增长战略回顾。

**本季度概况**：
- 开始时的目标：
- 实际达成情况：
- 主要增长举措：
- 重大事件或变化：

**请帮我进行战略层面的反思**：

## 1. 目标达成分析
- 目标达成度如何？差距原因是什么？
- 目标设定是否合理？
- 有哪些意料之外的成果？

## 2. 增长模式评估
- 本季度的增长主要靠什么驱动？
- 这种增长模式可持续吗？
- 是在建设「复利资产」还是在消耗「一次性资源」？

## 3. Loop 战略评估
- 核心 Loop 选择是否正确？
- Loop 的强度是否在提升？
- 需要增加新的 Loop 吗？

## 4. 能力建设回顾
- 在增长能力上有什么进步？
- 团队的增长意识有提升吗？
- 缺乏什么关键能力？

## 5. 下季度战略方向
- 保持什么？改变什么？
- 资源应该如何重新分配？
- 最重要的战略赌注是什么？
```

## 深度反思工具

### RP-301 | 增长卡点诊断

```markdown
我感觉增长遇到了瓶颈，请帮我诊断问题。

**症状描述**：
（描述你观察到的增长问题或停滞现象）

**已经尝试过的方法**：
（列出你尝试过但没效果的方法）

**请帮我诊断**：

1. **问题定位**
   请通过提问帮我精确定位问题：
   - 问题出在 Loop 的哪个环节？
   - 是效率问题还是规模问题？
   - 是 Loop 本身的问题还是执行的问题？

2. **根因分析**
   - 表面现象背后的根本原因可能是什么？
   - 之前的尝试为什么没效果？
   - 有没有遗漏什么重要因素？

3. **解决方向**
   - 针对根因，有哪些可能的解决方向？
   - 每个方向的优劣势是什么？
   - 推荐的优先尝试顺序是什么？

4. **快速验证方法**
   - 如何用最小成本验证解决方案是否有效？
   - 什么信号表明我们找到了正确方向？
```

### RP-302 | 决策复盘

```markdown
请帮我复盘一个重要的增长决策。

**决策背景**：
- 当时面临的情况：
- 有哪些选项：
- 最终做出的决策：
- 决策依据：

**决策结果**：
- 实际发生了什么：
- 与预期的差异：

**请帮我反思**：

1. **决策过程评估**
   - 当时的分析是否充分？
   - 有没有忽视什么重要信息？
   - 决策框架是否合理？

2. **结果归因**
   - 结果（好或坏）是因为决策正确还是运气？
   - 如果结果不好，是决策错误还是执行问题？
   - 有没有本可预见但没预见的因素？

3. **学习收获**
   - 这个决策教会我什么？
   - 对类似决策有什么启发？
   - 我的决策框架需要怎样更新？

4. **应用到未来**
   - 面对类似情况，下次会怎么做？
   - 有什么通用的原则可以提炼？
```

### RP-303 | 对话式深度反思

```markdown
我需要一次深度的对话式反思。请扮演一个资深的增长顾问和教练，通过对话帮我思考。

**我想反思的主题**：
[填写你想深入思考的问题或困惑]

**反思目的**：
[你希望通过这次反思获得什么]

**请求**：
1. 不要直接给我答案，而是通过提问引导我思考
2. 帮我看到我可能没注意到的角度
3. 挑战我的假设，但方式要建设性
4. 在对话过程中帮我梳理思路
5. 最后帮我总结对话的关键收获

让我们开始这次对话式反思。
```

## 使用建议

| 提示词 | 使用频率 | 最佳时机 |
|--------|----------|----------|
| RP-001 周度反思 | 每周 1 次 | 周五下午或周末 |
| RP-002 学习检查 | 每 2 周 1 次 | 完成一个学习阶段后 |
| RP-101 月度复盘 | 每月 1 次 | 月初（复盘上月） |
| RP-102 心智模型更新 | 每月 1 次 | 有重大实践经验后 |
| RP-201 季度回顾 | 每季度 1 次 | 季度末 |
| RP-301 卡点诊断 | 按需 | 感觉增长停滞时 |
| RP-302 决策复盘 | 按需 | 重要决策执行后 |
| RP-303 深度对话 | 按需 | 需要深入思考时 |

---

*记住：反思不是为了自责，而是为了学习和进步。*
