# Learning Path | 学习路径

## 学习目标

完成本学习路径后，你将能够：

1. **理解** Growth Loop 的本质，并清晰解释其与漏斗模型的差异
2. **识别** 任何业务中的核心增长环及其类型
3. **设计** 适合货代业务的增长飞轮
4. **量化** Loop 的关键指标并找到优化杠杆点
5. **执行** 系统性的增长实验并持续迭代

## 学习原则

1. **先理解 Why，再学习 What 和 How**
2. **每个概念都尝试用自己的业务举例**
3. **不懂的地方标记为「待深入」，但不阻塞学习进度**
4. **每完成一个 Phase，做一次整体回顾**
5. **实践产生的洞察优先级高于理论学习**
