# Analysis Prompts | 分析工具提示词

> 使用说明：这些提示词用于分析具体业务问题。在完成基础学习后使用效果最佳。

---

## Phase 5: [你的行业]场景分析

### AP-501 | [你的行业]特性分析

```markdown
请帮我系统性分析国际物流[你的行业]的商业特性，为后续设计 Growth Loop 做准备。

**我的背景**：我是一家国际物流[你的行业] 公司的创始人，主要业务是 [请填写：你的主营业务类型]，主要客户是 [请填写：[你的目标客户群]/你的客户类型]。

**请从以下维度分析**：

## 1. 行业价值链

- 国际物流的完整价值链结构是怎样的？
- 货代在价值链中的位置和角色
- 货代上下游的关系（船公司、报关行、拖车、仓库等）
- 货代创造价值的核心环节是什么？

## 2. 业务模式特性

- 货代的主要收入模式（差价/服务费/增值服务）
- 客户决策流程和采购行为是什么样的？
- 客户选择货代的关键因素排序
- 客户忠诚度特征（复购率、转换成本、生命周期）

## 3. 市场结构

- 国际货代市场的集中度如何？
- 头部企业 vs [你的行业] 中小企业的竞争格局
- 区域性 vs 全国性 vs 全球性货代的差异
- 数字化对行业格局的影响

## 4. 数字化现状

- [你的行业]数字化渗透的现状
- 数字货代（Digital Freight Forwarder）的发展趋势
- 传统货代的数字化转型方向
- SaaS 工具在[你的行业]的应用

## 5. 对 Growth Loop 设计的启示

基于以上分析：
- [你的行业]的哪些特性有利于建立 Growth Loop？
- 哪些特性是约束或障碍？
- 最有可能建立的 Loop 类型是什么？
- 需要特别注意什么行业特殊性？
```

**深入追问建议**：
- "[你的行业] 中小企业与头部货代的增长策略应该有什么不同？"
- "[你的行业]的客户获取成本（CAC）通常是什么水平？"
- "你的客户的推荐行为有什么特点？"

### AP-502 | 物流增长案例研究

```markdown
请帮我深入分析以下国际物流/供应链领域公司的增长策略。

**分析框架**：对于每家公司，请从以下维度分析

1. **业务模式概述**
   - 核心价值主张
   - 目标客户
   - 收入模式

2. **核心 Growth Loop 分析**
   - 他们的核心增长环是什么类型？
   - Loop 的具体结构（输入→行动→输出→再投入）
   - Loop 的关键转化节点
   - 周期时间估计

3. **冷启动策略**
   - 如何解决早期的获客问题？
   - 有没有网络效应？如何启动？

4. **增长杠杆点**
   - 他们优化了 Loop 的哪些环节？
   - 有什么独特的增长战术？

5. **可借鉴的策略**
   - 对中小[你的行业] 公司的启示
   - 可以直接借鉴的做法
   - 需要调整才能借鉴的策略

**请分析以下公司**：

### 案例 1: Flexport
- 作为数字货代的代表，他们的增长飞轮是什么？

### 案例 2: Freightos
- 作为货运定价/预订平台，他们如何解决双边市场的冷启动？

### 案例 3: Project44 / FourKites
- 作为物流可视化平台，他们的网络效应是如何建立的？

### 案例 4: Shippo
- 作为面向电商的物流 API 公司，他们的增长策略是什么？

**额外请求**：
如果有其他你认为值得研究的[行业科技]公司增长案例，也请补充。
```

**深入追问建议**：
- "[某公司] 的策略，[你的行业] 中小企业能复制吗？需要什么条件？"
- "这些公司有没有什么共同的增长模式？"
- "传统货代想数字化转型，应该学习哪家公司？"

### AP-503 | 我的业务增长画布

```markdown
请帮我为自己的[你的业务]设计增长画布。这是一个互动式的分析过程。

**我的业务基本信息**：
（请在使用时填写以下信息）

- 公司成立时间：
- 当前年收入规模：
- 员工人数：
- 主营业务：
- 主要航线/区域：
- 目标客户画像：
- 当前客户数量：
- 新客户主要来源：
- 客户复购率：
- 主要竞争对手：

---

**请帮我完成以下分析**：

## Part 1: 当前增长状态诊断

基于我提供的信息，请帮我诊断：
- 目前的主要获客方式属于哪种 Loop 类型？
- 这个 Loop 的大致效率如何？
- 目前增长的最大瓶颈可能是什么？

## Part 2: Loop 机会识别

对于我的业务，请分析：
- 哪些类型的 Loop 理论上可行？
- 每种 Loop 的潜在效果和实施难度？
- 优先级排序建议

## Part 3: 核心 Loop 设计

帮我设计一个最适合当前阶段的核心 Loop：
- Loop 的完整结构
- 每个环节的关键动作
- 需要追踪的核心指标
- 可能的启动策略

## Part 4: 实施路线图

- Phase 1（0-3 个月）：应该做什么？
- Phase 2（3-6 个月）：如何迭代优化？
- 关键里程碑和成功指标

## Part 5: 风险和约束

- 实施这个 Loop 的主要风险是什么？
- 需要什么资源和能力？
- 可能遇到的障碍和应对策略
```

**深入追问建议**：
- "如果我的销售团队只有 3 个人，这个 Loop 还可行吗？"
- "这个 Loop 需要什么样的技术/系统支持？"
- "如何设计第一个最小可行实验来验证这个 Loop？"

## 竞品分析提示词

### AP-601 | 竞品增长策略分析

```markdown
请帮我分析 [竞争对手名称] 的增长策略。

**竞品基本信息**：
- 公司名称：
- 主营业务：
- 规模（如知道）：
- 我观察到的他们的增长表现：

**请分析以下方面**：

1. **可观察的获客渠道**
   - 他们主要通过什么渠道获客？
   - 这些渠道的大致优先级
   - 有没有明显的创新做法？

2. **推测的 Growth Loop**
   - 基于可观察的信息，推测他们的核心 Loop
   - 这个 Loop 的可能效果
   - 他们的 Loop 有什么独特优势？

3. **内容与品牌策略**
   - 他们的内容营销做得如何？
   - 品牌定位和差异化
   - 客户口碑如何？

4. **产品/服务策略**
   - 他们的产品/服务有什么特别之处？
   - 是否有利于形成 Loop？

5. **我可以学习的点**
   - 直接可借鉴的策略
   - 需要避免的做法
   - 差异化机会在哪里？
```

## 增长实验设计提示词

### AP-701 | 增长实验设计助手

```markdown
我想设计一个增长实验来测试 [实验目标]。请帮我设计这个实验。

**实验背景**：
- 我想测试的假设是：
- 这个实验属于 Loop 的哪个环节：
- 我能投入的资源（时间/预算/人力）：

**请帮我设计**：

1. **实验假设**
   - 清晰的、可验证的假设陈述
   - 预期结果是什么

2. **实验设计**
   - 具体的实验方案
   - 实验组 vs 对照组（如适用）
   - 样本量建议

3. **成功指标**
   - 主要指标（1 个）
   - 次要指标（2-3 个）
   - 如何判断实验成功/失败

4. **实施步骤**
   - 具体的执行步骤
   - 时间表
   - 需要的工具或资源

5. **风险控制**
   - 可能出错的地方
   - 如何控制风险
   - 提前终止的条件

6. **学习计划**
   - 无论成功失败，我们能学到什么
   - 如何记录和分析结果
   - 下一步实验的方向
```
