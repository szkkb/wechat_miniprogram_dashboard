## 核心概念

### 1. Loop 的四个基本环节

| 环节 | 关键问题 | 核心指标 |
|------|----------|----------|
| **输入（Input）** | 新用户/客户从哪里来？ | 输入量 / 输入成本 |
| **动作（Action）** | 用户做了什么有价值的行为？ | 转化率 / 完成率 |
| **输出（Output）** | 产生了什么可被再投入的价值？ | 输出量 / 输出质量 |
| **再投入（Reinvest）** | 输出如何变成新的输入？ | 放大系数 / 再投入率 |

### 2. 复利 vs 线性增长

**为什么 "Loops compound, funnels don't"：**

假设每个周期你投入 100 个用户，转化率 50%，输出的用户又能带来 0.3 个新用户：

| 周期 | 漏斗模型（线性） | Loop 模型（复利） |
|------|------------------|-------------------|
| 1 | 100 | 100 |
| 2 | 100 | 100 + 15 = 115 |
| 3 | 100 | 115 + 17 = 132 |
| 4 | 100 | 132 + 20 = 152 |
| 5 | 100 | 152 + 23 = 175 |
| 6 | 100 | 175 + 26 = 201 |
| **累计** | **600** | **875** |

漏斗思维下你必须持续等量投入；Loop 思维下前期投入会产生叠加效应。

### 3. Cycle Time 与 Loop Efficiency

- **Cycle Time（周期时间）**：完成一轮 Loop 需要多长时间。周期越短，复利频率越高，增长越快。
- **Loop Efficiency（环效率）**：每轮 Loop 的输入到输出的转化比例。效率越高，放大系数越大。

> **关键洞察**：缩短 Cycle Time 通常比提升 Loop Efficiency 更有杠杆效应，因为它直接影响复利频率。

### 4. 强 Loop vs 弱 Loop 诊断

| 维度 | 强 Loop 信号 | 弱 Loop 信号 |
|------|-------------|-------------|
| 输入 | 输入持续增长，成本递减 | 输入依赖持续投入，成本递增 |
| 转化 | 用户自然完成关键行为 | 需要大量人工推动 |
| 输出 | 高质量、可量化的产出 | 产出模糊或不可度量 |
| 再投入 | 自动化、低边际成本 | 需要人工干预、高边际成本 |
| 周期 | 短（天/周级别） | 长（月/季度级别） |
